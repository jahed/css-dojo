# CSS Dojo

CSS Coding Dojo

## Installation

### Dependencies:
- Ruby

### Setup
```sh
git clone -b master --single-branch git@github.com:jahed/css-dojo.git
cd css-dojo
gem install bundler
bundle install
```

### Running

This project uses [Middleman](http://middlemanapp.com) for ease of use.

```sh
# Run local server at http://localhost:4567/
bundle exec middleman
```
